Sistem Operasi (Windows)

Sistem operasi ialah suatu perangkat lunak yang digunakan untuk mengoperasikan computer atau menjalankan kinerja komputer dan melakukan control serta menjalankan software aplikasi seperti program-program pengolah data yang digunakan untuk mempermudah kegiatan manusia. Serta sistem operasi Windows merupakan hasil perkembangan dari MS-DOS yakni sebuah sistem operasi yang berbasis modus teks dan command-line, dimana Windows berkembang mulai dari Windows 1.0 pada tahun 1985 sampai dengan Windows 10 pada tahun 2015.

Pengolah Kata (Word)

Microsoft Word merupakan bagian dari aplikasi Microsoft Office yang dikembangkan oleh perusahaan Microsoft, dimana aplikasi Microsoft Office terdiri dari dari Microsoft Excel, Powerpoint dan Word. Umunya aplikasi Microsoft Word digunakan untuk keperluan pekerja kantoran dan pelajar. Aplikasi Microsoft Word merupakan perangkat lunak untuk membuat lembar kerja berupa text dokumen dan sebagainya.

Pengolah Angka (Excel)

Microsoft Excel adalah bagian paket aplikasi Microsoft Office yang diluncurkan untuk keperluan pekerja kantoran bisnis dan pelajar. Microsoft Excel, merupakan sebuah program aplikasi lembar kerja yang dibuat dan di distribusikan oleh Microsoft Corporation. Fungsi utama Microsoft Excel ini adalah untuk mengolah angka, membuat tabel, serta menganalisis sejumlah data.

Pengolah Bahan Tayang (Google Slides)

Google Slides adalah salah satu aplikasi perkantoran yang berguna untuk menampilkan persentasi. Keunggulan aplikasi Google Slides ini adalah terletak pada kelebihan fiturnya yakni cloud bassed yang membuat Google Slides terhubung dengan cloud dalam penyimpanannya. Sehingga, seluruh perubahan yang dilakukan akan terekam secara real-time.

Desain Grafis (Figma)

Figma adalah tool desain berupa website yang terhubung dengan cloud sehingga bisa digunakan kapanpun dan dimanapun melalui internet. Tool ini berbasis Vector, sehingga akan lebih cocok untuk mendesain UI Website atau mobile dan aset ilustrasi. Figma juga bisa digunakan untuk mengedit foto walaupun hanya dengan pengaturan dasar saja. Figma mengutamakan kemudahan design jamming atau mendesain bersama-sama dalam satu platform tanpa harus menggabungkan secara manual.
